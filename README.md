**BERT-Based Phishing Detector: SMS Classifier and Anomaly Logic**

The BERT-Based Phishing Detector utilizes advanced machine learning techniques to classify SMS messages and identify potential phishing attempts. By analyzing patterns and employing anomaly detection, this tool enhances cybersecurity measures, safeguarding users from malicious threats and ensuring a more secure mobile environment.

**System Purpose**
To analyze SMS data and detect phishing attempts or OTP-based social engineering attacks.

**Techniques Used**
BERT-tiny fine-tuned for phishing classification
Keyword detection, OTP detection, URL parsing
Reason-explanation logic for transparency
SMS-call timeline fusion

**Anomaly Parameters**
confidence > 0.95 = high risk
OTP presence with unknown sender
Use of financial or urgent keywords
Domain-based blacklisting

**Unique Features (USP)**
NLP-based risk scoring
Confidence histogram
Interactive labeling and notes
PDF report with sender, domain, and risk charts
