**BERT-Based Phishing Detector: SMS Classifier and Anomaly Logic**

The BERT-Based Phishing Detector utilizes advanced machine learning techniques to classify SMS messages and identify potential phishing attempts. By analyzing patterns and employing anomaly detection, this tool enhances cybersecurity measures, safeguarding users from malicious threats and ensuring a more secure mobile environment.
